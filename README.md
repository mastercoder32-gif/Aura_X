## 📖 Overview

**Aura-X (AXS4)** is a sophisticated, minimalist security framework engineered for modern penetration testers and security researchers. It blends the power of industry-standard engines like **Nmap** and **Nuclei** with a custom-built logical core, delivering a seamless "Quiet Luxury" terminal experience.

Designed with **legal compliance** and **professional ethics** at its heart, Aura-X ensures that your reconnaissance is precise, documented, and high-performance.

---

## 🔥 Key Features (Basic Edition)

- **⚡ Stealth Service Discovery:** Advanced Nmap integration for fingerprinting services without triggering aggressive alarms.
- **🔍 Deep Vulnerability Mapping:** Real-time CVE scanning using the latest YAML-based templates.
- **🛡️ Legal Guard Module:** Integrated `Disclaimer.py` ensures all operations are bound by ethical hacking standards.
- **🎨 Premium UI/UX:** A minimalist terminal interface with gold-accented status bars and fluid progress tracking.
- **🚀 Dependency Auto-Sync:** Automated checks for required binaries and Python libraries.
- **📊 Detailed Logging:** Generates clean, actionable insights for every scanned target.
